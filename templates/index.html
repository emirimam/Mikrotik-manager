🔗 RoMON – Remote Device Access
How RoMON works in this app
1. Set up your proxy router (with public IP) in the left sidebar
2. Click Scan Neighbours to discover devices reachable via RoMON
3. Click a device to select it as target
4. Go to 📜 Script tab → check Deploy via RoMON
5. Deploy your config script through the tunnel

⚠ Requirement: The proxy MikroTik must have RoMON server enabled and SSH port 22 open.
